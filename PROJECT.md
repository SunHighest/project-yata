# Project YATA 憲章

> **Technology should feel natural.  
> AI should feel like a trusted partner.**
>
> **テクノロジーは自然であるべき。  
> AIは信頼できる相棒であるべき。**

---

## Project YATAが目指すもの

Project YATAは、人間を置き換えるAIを作るプロジェクトではありません。

人がより良く考え、学び、創造し、行動するための「相棒」を創る研究開発プロジェクトです。

私たちは、AIとの対話がチャットではなく、日常会話のように自然である未来を目指します。

音声を中心に、必要なときだけ視覚的な情報を提示し、人とAIが自然に協力できる体験を実現します。

Project YATAは、一時的な流行ではなく、10年後、20年後も価値を持つAIパートナーを目指して開発を続けます。

---

### Project YATA Principles

- 人間が目的を決める。
- AIは最適な方法を提案する。
- AIは人間の思考を奪わず、能力を拡張する。
- シンプルさを最優先する。
- 完成を目指さず、継続して進化する。
- AIに任せられることはAIに任せる。
- 人間は創造と決断に集中する。

# プロジェクト概要

Project YATA は、人と共に考え、学び、行動するAIパートナーを実現するための研究開発プロジェクトです。

映画に登場するJARVISのような体験を目指しますが、単なるチャットAIではありません。

音声による自然な対話を中心とし、必要なときだけ画面で視覚的に情報を提示する、人に寄り添うAIの実現を目標とします。

---

# YATAの由来

YATAは日本神話に登場する「八咫烏（やたがらす）」から名付けました。

八咫烏は「導き」の象徴とされ、人々を正しい方向へ導く存在です。

Project YATAも、人間の代わりに判断するAIではなく、人間がより良い判断をできるよう導くAIを目指します。

現在の略称

YATA

- Yielding
- Autonomous
- Thinking
- Assistant

※ Y の意味は今後変更する可能性があります。

---

# ビジョン

Project YATAは段階的に進化します。

### Phase 1
音声によるAIパートナー

### Phase 2
必要な時だけ画面で説明するAI

### Phase 3
PC操作や仕事を支援するAI

### Phase 4
Physical AIとの連携

最終的には、人が毎日自然に会話したくなるAI相棒を目指します。

---

# 基本理念

YATAは人間の代わりになるAIではありません。

人間の能力を拡張するAIです。

- 最終判断は人間
- 思考整理はYATA
- 情報収集はYATA
- 自動化はYATA
- 創造は人とAIが協力して行う
